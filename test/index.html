사용 가이드

  • ⭐ 개인화 즐겨찾기 카드
    원하는 URL을 직접 추가/삭제하여 본인만의 카드로 활용할 수 있습니다.

  • 🖱️ 카드 순서 변경
    마우스로 드래그하여 카드 순서를 원하는 대로 재배치할 수 있습니다.

  • 🔒 카드 고정/해제
    카드 고정 → 순서 변경 불가 / 카드 해제 → 순서 변경 가능

  • 🙈 카드 숨김
    각 카드 우측 상단의 ✖ 버튼을 눌러 카드를 숨길 수 있습니다.
    숨긴 카드는 상단의 ‘숨김 카드 목록’에서 다시 복원할 수 있습니다.

  • 📝 메모장 카드
    빈 메모장 카드를 활용해 간단한 메모를 기록할 수 있습니다.
    입력한 메모는 자동 저장되어 새로고침 후에도 유지됩니다.

  • 🔍 검색 접두어
    검색창에 g 또는 ㅎ 또는 ㄱ 입력 후 띄어쓰기 → 구글 검색 /
    n 또는 ㅜ 또는 ㄴ 입력 후 띄어쓰기 → 네이버 검색

  • 📑 카드 필터링
    상단 필터 버튼을 눌러 업무·수업·외부 자료 등 원하는 카드만 볼 수 있습니다.

  • 📱 스마트폰 사용
    모바일에서도 동작하지만, PC 환경에 최적화되어 있습니다.
